About

This web application was created by Magda Costa, Rafael Pacheco and Sofia Machado and it illustrates the use of the Disney Movies database. The application is built using sqlite3, Flask, and Jinja.

Through various SQL queries and using techniques we learned throghout this semester, we managed to build this SQl embedded application which will allow you to search relevant data regarding our dataset more efficiently.
